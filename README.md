# 📊 College Major Salary Analysis (Python)

## 📌 Overview

This project analyzes salary data across different college majors to understand earning potential, career growth, and risk associated with different fields.

---

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib

---

## 📊 Analysis Performed

* Data cleaning and preprocessing
* Salary comparison across majors
* Risk vs reward analysis using salary spread
* Group-wise salary analysis

---

## 📈 Visualizations

* Top 10 highest paying majors
* Salary spread comparison
* Average salary by major group

---

## 🔍 Key Insights

* Engineering majors dominate high-paying careers
* Some majors have high growth but also higher risk
* STEM fields have the highest average salaries
